Documentation for the Netiom Library
Module Properties
 



[This is preliminary documentation and is subject to change.]

A Module represents a single card. It has no public constructor. It is returned from Netiom.GetModule.

The following tables list the members exposed by the Module type.

Public Properties

  NameDescription
Item
Gets the specified input port or Sets the specified output port.

ModuleHostName
Gets the hostname of the module.

ModuleIPAddress
Gets the module IP address.

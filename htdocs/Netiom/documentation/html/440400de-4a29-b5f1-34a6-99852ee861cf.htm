SetPorts(Boolean[]())
Sets any of the output ports, starting from 1 with a call that looks like: SetPorts(false, false, true, false).
SetPorts(String)
Sets any of the output ports, starting from 1 from a string comprising 1s, 0s and *s If the string length is less than 16, it will be padded with *s.
SetPorts(UInt16)
Sets all the output ports.

Documentation for the Netiom Library
MayBE.Netiom Namespace
 

[This is preliminary documentation and is subject to change.]

Utilities for the Netiom UDP card.

Classes

  ClassDescription
Module
A Module represents a single card. It has no public constructor. It is returned from Netiom.GetModule.

Netiom
Main static class. This waits for a message from a card and then creates a module and starts a heartbeat to it.

Delegates

  DelegateDescription
AnalogChange
Callback routine for a change to a analog port.

PortChange
Callback routine for a change to a digital port.

Enumerations

  EnumerationDescription
IO
The port set we are interested in.

Documentation for the Netiom Library
Module Fields
 



[This is preliminary documentation and is subject to change.]

A Module represents a single card. It has no public constructor. It is returned from Netiom.GetModule.

The following tables list the members exposed by the Module type.

Protected Fields

  NameDescription
AnalogPorts
4 delegates for the 4 output ports.

device
IP endpoint of the module.

hostname
Hostname of the module. Completed when Hostname first called.

InputPorts
16 delegates for the 16 input ports.

IPaddress
IP address of the module.

lastAnalog
State of the analog ports as last seen.

lastInputs
State of the input ports as last seen.

lastOutputs
State of the output ports as last seen.

OutputPorts
16 delegates for the 16 output ports.
